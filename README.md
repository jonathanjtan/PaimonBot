# PaimonBot
Genshin Impact resource tracking bot.

create .env and populate with DISCORD_TOKEN and DISCORD_SERVER

e.g.
```
DISCORD_TOKEN=$SERVER_TOKEN
DISCORD_SERVER=$SERVER_NAME
```
